## Worker 架构

```
main (打工人首领)
  │
  ├── product-worker (产品君) 📋
  │   └─ 需求分析、PRD 编写
  │
  ├── ui-worker (UI君) 🎨
  │   └─ 界面设计、交互流程
  │
  ├── web-worker (Web君) 💻
  │   └─ Web 前端开发
  │
  ├── app-worker (App君) 📱
  │   └─ 移动端开发
  │
  └── qa-worker (QA君) 🧪
      └─ 测试、质量报告
```

---

## 协作流程

```
用户请求 → main 分析
    │
    ├─ 产品需求 → product-worker
    ├─ UI 设计 → ui-worker
    ├─ Web 开发 → web-worker
    ├─ App 开发 → app-worker
    └─ 测试验证 → qa-worker
         │
         ▼
    结果汇总 → main 整合 → 用户
```
