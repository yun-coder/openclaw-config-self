#### 会话压缩策略 (compact.rs)
- **配置**: preserve_recent=4, max_tokens=10,000
- **流程**: 检查 → 保留最近 → 生成摘要 → 合并历史
- **摘要内容**: Scope, Tools, Requests, Pending work, Key files, Timeline

#### 工具权限粒度 (tools/lib.rs)
- **5 级权限**: ReadOnly → WorkspaceWrite → DangerFullAccess → Prompt → Allow
- **每个工具声明**: `required_permission`
- **示例**: read_file=ReadOnly, bash=DangerFullAccess

#### Token 成本追踪 (usage.rs)
- **数据**: input, output, cache_write, cache_read
- **定价**: 按模型区分 (Haiku/Sonnet/Opus)
- **追踪器**: latest_turn, cumulative, turns

#### 项目上下文发现 (prompt.rs)
- **层级发现**: 从当前目录向上查找 CLAW.md
- **Git 注入**: status + staged diff + unstaged diff

#### 钩子系统 (hooks.rs)
- **退出码**: 0=允许, 2=拒绝, 其他=警告
- **环境变量**: HOOK_EVENT, HOOK_TOOL_NAME, HOOK_TOOL_INPUT
- **Payload**: JSON stdin

## 关键学习点

### 会话压缩智能摘要
```
<summary>
Conversation summary:
- Scope: 10 earlier messages compacted (user=3, assistant=4, tool=3).
- Tools mentioned: read_file, edit_file, bash.
- Recent user requests:
  - Update the compaction logic
- Pending work:
  - Next: update tests
- Key files referenced: src/compact.rs.
- Current work: Working on coverage.
- Key timeline:
  - user: Update the compaction logic
  - assistant: I will inspect the flow.
</summary>
```

### 工具权限映射
| 工具 | 权限 |
|------|------|
| read_file, grep, glob, WebFetch, WebSearch, Skill | ReadOnly |
| write_file, edit_file, TodoWrite | WorkspaceWrite |
| bash, Agent | DangerFullAccess |

### Token 定价
| 模型 | Input | Output | Cache Write | Cache Read |
|------|-------|--------|-------------|------------|
| Haiku | $1/M | $5/M | $1.25/M | $0.1/M |
| Sonnet/Opus | $15/M | $75/M | $18.75/M | $1.5/M |
