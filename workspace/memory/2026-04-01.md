## 创建的钩子

### tool-permission-check

```
位置: hooks/tool-permission-check/
功能: 在工具执行前检查权限级别
权限分级:
  - ReadOnly: read, web_search, memory_search
  - WorkspaceWrite: write, edit, image_generate
  - DangerFullAccess: exec, process, sessions_spawn
```

### cost-alert

```
位置: hooks/cost-alert/
功能: 在会话成本超过阈值时发出提醒
阈值:
  - ¥0.10: 无提醒
  - ¥0.30: 提示
  - ¥0.50: 建议压缩
  - ¥1.00: 建议新会话
```

---

## 启用钩子

```bash
# 启用工具权限检查
openclaw hooks enable tool-permission-check

# 启用成本警告
openclaw hooks enable cost-alert

# 检查钩子状态
openclaw hooks list
```
